# 🐾 桌面宠物助手 — 全景设计方案

## 一、产品定位

> **一只住在桌面上的 AI 小精灵** — 既是陪伴型桌面宠物，也是智能工作助手。

它不是冰冷的工具栏，而是一个有生命感的存在：会打盹、会卖萌、会提醒你喝水、也会帮你搜索资料、整理笔记、管理日程。**情感陪伴 + 功能助手** 二合一。

---

## 二、角色系统设计

### 2.1 默认角色：小光（Lumie）

**设计理念**：圆润 + 微光感，既可爱又带点科技感。「小精灵」风格，适配各种场景。

### 2.2 角色皮肤系统

| 皮肤 | 风格 | 解锁条件 |
|------|------|----------|
| 小光（默认） | 精灵/微光 | — |
| 煎蛋仔 | 食物/搞怪 | 连续签到7天 |
| 黑猫 | 神秘/慵懒 | 开启夜间模式累计50小时 |
| 机器人 | 赛博/硬核 | 使用开发者工具累计100次 |
| 自定义 | 用户上传 | 高级版 |

### 2.3 表情状态机

## 四、交互设计

| 操作 | 效果 |
|------|------|
| **单击宠物** | 唤醒 + 问候语 + 打开对话气泡 |
| **双击宠物** | 打开完整助手面板 |
| **长按拖拽** | 移动宠物位置（带物理弹性） |
| **右键** | 快捷菜单（笔记/提醒/设置） |

### 智能主动行为

| 场景 | 宠物行为 | 频率 |
|------|---------|------|
| 连续工作1小时 | "该喝水啦～" | 每小时1次 |
| 到了饭点 | "饿了饿了好饿啊" | 12:00/18:00 |
| 长时间无操作 | 自娱自乐 | 闲置15分钟 |
| 日历有会议 | "5分钟后有会议" | 会议前5分钟 |
| 深夜还在工作 | "早点睡嘛…" | 23:00后 |

## 五、技术架构

- **Tauri 2.0**（Rust 壳层，5MB 安装包，40MB 内存）
- **WebView 渲染层**：Pet Canvas（动画引擎）+ UI Overlays
- **AI 服务层**：本地 Ollama / 云端 Claude API / 通义
